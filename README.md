# What is Wordsy?

Wordsy is a minimal, intuitive notes and to-do app designed to help you stay organized without overwhelming you.\
It focuses on clarity, speed, and a smooth writing flow, making it easy to record anything the moment it comes to mind.

#### Experience it Yourself!

This initial version sets the foundation for future updates, emphasizing stability, simplicity, and a delightful note-taking experience.\
Try it out and feel how effortless it is to write, remember, and stay organized with Wordsy.
